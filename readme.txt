Video embed filter
--------------------------

Make it easy to embed youtube or vimeo videos into page content using their HTML5/Flash api

To add a video to a page:
    Edit the page and add a piece of text like this:
    [[videoembed:video url]]
    eg [[videoembed:http://www.youtube.com/v/fYTJ9v2vsaE]]

Will accept the url as either the url from your browser
    eg [[videoembed:http://www.youtube.com/watch?v=fYTJ9v2vsaE&mode=related&search=]]
